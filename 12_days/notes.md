\# React Basic



\## Component

\- React applications are divided into small parts called components

\- Components are reusable pieces of UI



\## Functional Component

\- A component created using a JavaScript function

\- It returns JSX



\## JSX

\- JSX allows writing HTML-like syntax inside JavaScript

\- Used to create UI in React



\## Import

\- Used to bring code from another file



\## Export

\- Used to make a file available for use in other files





\# Event Handling Notes



\## Event

\- User actions are called events

\- Examples:

&#x20; - Click

&#x20; - Mouse Enter

&#x20; - Mouse Leave

&#x20; - Input Change



\## onClick

\- Runs a function when a button is clicked



\## onMouseEnter

\- Runs a function when the mouse enters an element



\## onMouseLeave

\- Runs a function when the mouse leaves an element



\## onChange

\- Runs a function when the input value changes



\## Event Object

\- React automatically provides an event object

\- It contains information about the event





\# useState Notes



\## useState

\- A React Hook used to manage state



\## State

\- Dynamic data that can change over time

\- Updating state re-renders the UI



\## Initial State

\- The starting value passed into useState



\## State Update Function

\- Function used to update the state value



\## Previous State

\- Accessing the old state value before updating





\# Props Notes



\## Props

\- Used to pass data from parent component to child component



\## children Prop

\- Represents the content placed between opening and closing component tags





\# Reusable Components



\## Reusable Component

\- A component that can be used multiple times

\- Helps reduce code duplication





\# Arrow Function Notes



\## Arrow Function

\- Modern JavaScript function syntax

\- Cleaner and shorter syntax





\# Tailwind CSS Notes



\## Tailwind CSS

\- A utility-first CSS framework

\- Styling is done using utility classes



\## className

\- React uses className instead of class





\# React Data Flow



\## Parent Component

\- Component that sends data



\## Child Component

\- Component that receives data



\## One Way Data Flow

\- Data flows from parent to child in React





\# Counter Logic Notes



\## Increment

\- Increasing a value



\## Decrement

\- Decreasing a value



\## Reset

\- Returning a value to its initial state



\## Dynamic Value Change

\- Updating state with different values





\# React Application Structure



\## App Component

\- Main/root component of the application



\## Component Folder

\- Used to organize components properly
